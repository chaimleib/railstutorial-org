== README

This a tutorial project, following the book at https://www.learnenough.com/ruby-on-rails-4th-edition-tutorial/beginning.

The latest version of the book is at https://www.railstutorial.org, and is paywalled.

To init:

```sh
bundle install
```

To run:

```sh
rails server -u puma
```

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
